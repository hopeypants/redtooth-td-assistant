Redtooth TD Assistant

Supercharge your Redtooth venue admin workflow with helpers for Tournament Directors — Edit Scores, lists, add-player tools, and more. Built for redtoothpoker.com/venue_admin.

Chrome extension Manifest V3 Open source
Get started View on GitHub

Features

Edit Scores

  • Season week navigation (past / future)
  • Adjustable table row spacing
  • Floating “Update” button (position left or right)
  • Player archive & live name filter (with optional Ctrl/Cmd+F focus)
  • Duplicate rank detection: highlight, prevent, or off
  • Optional filter to show only duplicate-rank rows

Lists & dates

  • List Season Score Weeks: friendlier date formats
  • Optional “venue play day” column

Add Player

  • Membership number tools & optional field highlights
  • Default address values when enabled

Options & UX

  • Full options page (light / dark / system theme)
  • Optional sync of settings across Chrome profiles
  • “Loaded” badge on venue admin (with extension icon)

Browser compatibility

Works on Chromium-based browsers with Manifest V3 support:

  • Google Chrome
  • Microsoft Edge
  • Brave
  • Opera
  • Vivaldi

Installation (from source)

This extension is installed in developer mode from a local build. You need Node.js 18+.

  1. Get the code
    Option A: On GitHub, click Code → Download ZIP, then extract it.
    Option B: Clone with Git: 
    git clone https://github.com/hopeypants/redtooth-td-assistant.git
cd redtooth-td-assistant
  2. Install dependencies & build (creates the dist/ folder): 
    npm install
npm run build
  3. Load in Chrome: open chrome://extensions → enable Developer mode → Load unpacked → select the dist folder (not the repo root).
  4. Other Chromium browsers: use their extensions page (e.g. edge://extensions) the same way.
Tip: Pin the extension from the puzzle-piece menu so the toolbar icon and Options are easy to reach.

Updating

  1. Pull or re-download the latest code, then from the project folder run 
    npm install
npm run build
  2. Open chrome://extensions, find Redtooth TD Assistant, and click Reload (circular arrow).

Settings live in chrome.storage and usually persist across reloads. If you use sync, updates follow your Google account like other extensions.

Usage

  • Visit Redtooth venue admin as usual (Tournament Director area).
  • Open Options from the extension icon to turn features on or off and tune behaviour (Edit Scores, floating Update, duplicate ranks, archive, etc.).
  • Content scripts only run on https://www.redtoothpoker.com/venue_admin/* — there is nothing to configure on unrelated sites.

Privacy

Settings and optional archive data stay in your browser via chrome.storage. The extension does not send your venue or player data to third-party servers — it only enhances pages you already open on Redtooth.
