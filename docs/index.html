Redtooth TD Assistant

Install the Chrome extension from source — helpers for Redtooth venue admin (Tournament Director).

Prerequisites

Node.js 18+ and Google Chrome (or Chromium: Edge, Brave, etc.).

Install (unpacked)

  1. Clone the repository (or download a ZIP from GitHub and extract it).
  2. In a terminal, from the project folder: 
    npm install
npm run build
    This creates the loadable extension in the dist folder.
  3. Open Chrome and go to chrome://extensions.
  4. Turn on Developer mode (top right).
  5. Click Load unpacked and select the dist directory (inside your project folder — not the repository root).
  6. Optional: pin the extension and open Options from the toolbar to configure features.

After git pull or other updates, run npm run build again, then use Reload on the extension card in chrome://extensions.

Privacy

Settings are stored with chrome.storage. The extension runs on Redtooth venue admin pages only; it does not send your venue data to third-party servers.
